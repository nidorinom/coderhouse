Arcanos Mayores

El proyecto Arcanos Mayores consiste en la reinterpretación de las cartas de Tarot, donde se toma como punto de partida los artes originales de las barajas más famosas del Tarot como lo es el Tarot de Marsella y el trabajo de Pamela Waite Smith, para crear una nueva versión siguiendo estilos propios e interpretaciones personales.

La técnica usada es dibujo digital, siendo Adobe Illustrator la herramienta usada para la creación de estas piezas.
